Bun.js in Browser Demos

Run Bun.js-style servers in the browser with a WebSocket-based reverse proxy. This project allows you to create and run Bun.js servers directly in your web browser, enabling server-like functionality without the need for a traditional backend.

View on GitHub

Features

  • Run Bun.js-style servers in the browser
  • WebSocket-based communication between browser and server
  • Support for various HTTP methods (GET, POST, etc.)
  • JSON response handling
  • HTML response generation
  • Form data processing
